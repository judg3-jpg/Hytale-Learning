## Step 1: Build the Plugin

```bash
cd /path/to/minecraft-plugin
mvn clean package
```

The JAR file will be at: `target/MegaWalls-1.0.0.jar`

---

## Step 2: Install

1. Copy `MegaWalls-1.0.0.jar` to your server's `plugins/` folder
2. Restart your server

---

## Step 3: Quick Setup (In-Game)

As an OP/admin player:

```
/mwadmin quicksetup
```

This sets up everything automatically for testing!

---

## Step 4: Play!

### Join the game:
```
/mw join
```

### Pick a class:
```
/mw class hero      # Support with healing
/mw class marksman  # Ranged sniper
/mw class warrior   # Melee berserker
```

### Start the game (admin):
```
/mw start
```

---
