## How to View

1. **Start the server** (if not running):
   ```bash
   npm start
   ```

2. **Open the dashboard**:
   - Go to `http://localhost:3000`
   - Or click the extension icon

3. **View the cards**:
   - All 14 moderator cards should be visible
   - Each card shows:
     - MODERATOR tag
     - Moderator name
     - Work section
     - Current month stats
     - Performance chart
     - Status badge

## Card Features

- **Clickable**: Click any card to view detailed statistics
- **Hover effects**: Cards lift and glow on hover
- **Responsive**: Works on all screen sizes
- **Dark mode**: Automatically adapts to theme

## To Add More Moderators

Run the seed script again (it will update existing ones):
```bash
npm run seed
```

Or manually add via the "Add Moderator" button in the dashboard.
